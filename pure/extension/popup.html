Chrome Extension Template

Popup Page

You are on "" page.
It reads the title of the current page by using chrome.tabs.query
Open Google in a new tab
It opens a new tab and visits Google by using chrome.tabs.create
Send a message to background.js
It sends a message to background.js in order to set a short text on the badge(over the icon) by using chrome.runtime.sendMessage
Send a message to content.js and then send an XHR request to your API from popup page
In this case it will first sends a message to content.js and then the content.js will send an XHR request to your api let's say api.github.com to fetch a repository. content.js will get the response from api and send it back to popup page. Right click here > "inspect" to see the responded data in console. to see the XHR request, open Devtools > Network on the active tab.
Inject an HTML object into current tab
In this case it will first sends a message to content.js and then the content.js will inject a simple html object inside the current tab. You will see "Feedback" container on the right buttom.
